Contact Grabber Options

Destination:

Select where to deliver grabbed contact
Hot Key: Alt+Shift+PageDown
Download CSV file
Download VCard file
Upload to web site
Available Destination URLs:
  • One URL per line
  • First URL is active selection
  • Add optional label at end of each URL after ## (e.g., http://my.domain.com/path?myid=123##My Domain [123])
Active Destination URL:
Go to new tab after posting contact
 

Lookup:

Select site to lookup grabbed contact
Hot Key: Alt+Shift+PageUp
Available Lookup URLs:
  • One URL per line
  • First URL is active selection
  • Supply substitution variables of the form %FIELD% (only name, email, phone, and URL fields supported)
  • Add optional label at end of each URL after ## (e.g., http://my.domain.com/path?name=%NAME%##My Domain [by name])
